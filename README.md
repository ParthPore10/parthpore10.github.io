# Portfolio Website (HTML/CSS/JS)

Single-page personal portfolio built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Edit content

Update details in:

- `index.html` (text, links, projects, experience)
- `style.css` (visual theme, spacing, typography)
- `script.js` (simple reveal animation + footer year)

## Theme + Photo

- Click the top-right toggle to switch light/dark mode.
- Profile image path: `assets/my_photo.jpeg`
